# Homebrew Tap for bits CLI

Homebrew formulae for [bits CLI](https://github.com/mdnmdn/bits).

## Install

```sh
brew install mdnmdn/bits/bits
```

Or tap first, then install:

```sh
brew tap mdnmdn/bits
brew install bits
```

## More info

See the [main repository](https://github.com/mdnmdn/bits) for documentation and usage.
